# HTMLの基本構造と必須タグ
`<!DOCTYPE html>` から `<body>` までの骨組みと、よく使うコンテンツタグを押さえます。

## 本章の目標

本章では以下を目標にして学習します。

- HTML文書の最小構成を自分で書けること
- `meta charset` の意味を説明できること
- 見出し・段落などの基本タグを適切に使えること

## 1. 最小の HTML 文書

```html
<!DOCTYPE html>
<html lang="ja">
<head>
  <meta charset="utf-8">
  <title>ページのタイトル</title>
</head>
<body>
  <h1>メインの見出し</h1>
  <p>本文の段落です。</p>
</body>
</html>
```

- `<!DOCTYPE html>` … ブラウザに「HTML5だよ」と伝える宣言  
- `<html lang="ja">` … 文書の言語（アクセシビリティ・検索に効く）  
- `<head>` … 画面に直接出さない情報（文字コード、タイトル、CSSの読み込みなど）  
- `<body>` … 画面に表示する本体  

> **ポイント**  
> `<meta charset="utf-8">` は**文字化け防止**のためほぼ必須です。`<head>` のできるだけ早い位置に書くのが推奨されます。

## 2. 見出しと段落

- `<h1>` 〜 `<h6>` … 見出し（**ページに主として1つの h1** が推奨されることが多い）  
- `<p>` … 段落  

```html
<h2>小見出し</h2>
<p>1つ目の段落。</p>
<p>2つ目の段落。</p>
```

## 3. コメントとインデント

```html
<!-- これはコメントです。ブラウザには表示されません -->
```

インデントは動作に影響しませんが、**入れ子の構造が分かりやすい**ように揃えると保守しやすいです。

## まとめ

- `<!DOCTYPE html>` と `html` / `head` / `body` で文書の枠が決まる  
- `meta charset="utf-8"` で**文字コード**を明示する  
- 見出しは `h1`〜`h6`、本文のかたまりは `p` で表す
