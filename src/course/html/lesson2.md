# HTMLの役割と記述方法
HTMLの基本的な書き方を理解します。

## 本章の目標

本章では以下を目標にして学習します。

- HTMLの役割を理解できること
- HTMLの記述方法を説明できること

## 1. HTMLとは

HTMLの正式名称は「Hyper Text Markup Language」といいます。  
HTMLを学ぶときによくある勘違いがWebページを「デザインする」ためのものだと考えてしまうことです。  
HTMLは構造を記述するためのもので、どこに何を配置するのかをあらかじめ用意された文法に従って記述します。

## 2. 要素とタグ
HTMLでは`<○○>`というような記述が多く出てきます。  
これは「タグ」と呼ばれるもので、HTMLの基本的な要素を記述していくためのものです。

HTMLではテキストや写真をはじめ、入力フォーム、テーブル、リストなどさまざ要素を記述するためのタグが用意されています。  
どのような役割のタグがあるかわからないと思い通りのWebページを作ることはできません。  
まずはタグの使い方を覚えましょう。

## 3. タグの記述方法
まずは代表的な`<h1>`というタグを見てみましょう。  
このタグはタイトルなど「ページに何が書かれているか」を把握するための要素に使用されます。  
この`<h1>`は以下のようにように「開始タグ」(`<○○>`)と終了タグ(`</○○>`)で記述します。  
```html
<h1>タイトルです。</h1>
```

この他に、1つのタグしかないタグもあります。  
例として`<br/>`という改行するためのタグがあります。
このタグ以下のように`<○○>`や`<○○/>`のように記述します。
```html
<br> または <br/>
```  

## まとめ

- HTMLではタグを使用して構造を作る
- タグは「開始タグ」(`<○○>`)と終了タグ(`</○○>`)で記述する
- タグの中には`<○○>`や`<○○/>`のように1つのタグで記述するものもある
