# HTMLハンズオン
配布教材をベースにマークアップを追記し、質疑応答で仕上げます。

## 本章の目標

本章では以下を目標にして学習します。

- 講義で学んだタグを**実際のファイル**に反映できること
- ブラウザでの表示と HTML の対応を確認できること
- 分からない点を質問に落とし込めること

## 1. 進め方（想定）

- **0から100まで一気に作らず**、配布した HTML に**追記・修正**していく形式を想定しています。  
- 講師のデモに合わせて、見出し・リスト・リンク・画像などを足していきます。  

> **ポイント**  
> 手を動かした直後に「なぜこう表示されたか」を言葉にすると、定着が早くなります。

## 2. 確認のコツ

- ファイル保存後、ブラウザを**再読み込み**する  
- 意図と違うときは、**タグの閉じ忘れ**・**入れ子のズレ**を疑う  
- 開発者ツールの **Elements** で DOM を見る（詳細は後のデバッグの章でも扱います）  

## 3. 質疑応答

想定では**約5分**程度を質問に充てます。  
詰まっている行・エラーメッセージ・画面の状態を共有できるようにしておきましょう。

## まとめ

- ハンズオンは**配布ベースの追記**で時間内に完了しやすい  
- 表示と HTML の対応を**都度確認**する  
- 質問は**具体的な状態**付きで出す
