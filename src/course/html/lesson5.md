# その他のタグ
これまでの章で文章の骨組みを学びました。本章では、Webページをさらに豊かにする画像やリスト、入力フォームなどの頻出タグを学習します。

## 本章の目標

本章では以下を目標にして学習します。

- 画像、リンク、リスト、表（テーブル）、入力フォームのタグを適切に使い分けられること
- タグに情報を付加する「属性」の概念を理解すること

## 1.  `<img>`
Webページに画像を表示するには、`<img>`タグを使用します。  
`<img>`タグは終了タグを持たない単独のタグです。  
画像を表示するためには、どこにある画像かを示す`src`属性（ソース）と、画像が表示されない時や音声読み上げ用のテキストを示す`alt`属性（代替テキスト）を必ず指定します。  

```html
<img src="./images/sample.png" alt="カフェの外観写真">
```
ここで最も重要なのが src 属性に記述する**パス（Path）**です。  
パスとは、コンピュータ内での「ファイルの住所」のことで、相対パスと絶対パスが存在します。  
- 相対パス
「今いるファイル（自分）」を基準にして、目的のファイルの場所を示す方法です。
以下のように記述します。  
`./` ：今いる場所と同じフォルダ
`../`：一つ上の階層のフォルダ
- 絶対パス  
`https://~` から始まるURLのように、場所を特定する完全な住所です。 
   
開発現場では、画像やリンクの指定に「相対パス」を多用します。  
srcを書いたのに画像が表示されない原因のほとんどはパスの記述ミスです。  
**自分が今どの階層のファイルを作業しているのかを常に意識しましょう。**  

## 2. `<a>`
他のページへ移動する「リンク」を作るには`<a>`タグ（アンカータグ）を使用します。  
移動先のURLはhref属性で指定します。  

```html
<a href="https://example.com">ここをクリックして詳細を見る</a>
```

## 3. `<ul>` `<ol>` `<li>`  
情報を箇条書きでまとめる際は、リストタグを使用します。用途に合わせて2種類を使い分けます。  
`<ul>`タグは順番が関係ない箇条書きで使用され、・～と表示されます。  
`<ol>`タグは手順など、順番に意味がある箇条書きで使用され、1.～と表示されます。  
リストの各項目には、どちらの場合も`<li>`タグを使用します。  

```html
<ul>
  <li>HTML</li>
  <li>CSS</li>
  <li>JavaScript</li>
</ul>

<ol>
  <li>エディタを開く</li>
  <li>コードを書く</li>
  <li>ブラウザで確認する</li>
</ol>
```

## 4. `<table>`  
データを表形式で分かりやすく表示するには、`<table>`タグといくつかのタグを階層構造にして使用します。  
テーブルを作成するには`<table>`タグで表全体を囲みます。  
`<tr>`で表の「行」（横一列）を作ります。  
`<th>`で見出しのセルを作ります。  
`<td>`でデータセルを作ります。  
```html
<table>
  <tr>
    <th>名前</th>
    <th>年齢</th>
    <th>職業</th>
  </tr>
  <tr>
    <td>山田太郎</td>
    <td>28</td>
    <td>エンジニア</td>
  </tr>
  <tr>
    <td>佐藤花子</td>
    <td>25</td>
    <td>デザイナー</td>
  </tr>
</table>
```  

## 5. `<form>`  
ユーザーがテキストを入力したり、ボタンを押したりするためのフォームを作るには、`<form>`タグと`<input>`などのタグを組み合わせます。  
`<form>`でフォーム全体を囲みます。  
`<label>`で`<input>`の名称を記述します。  
`<input>`でテキストの入力欄や、チェックボックスなどを作ります。  
`<button>`タグで送信ボタンなどを作ります。  
`<input>`タグはtype属性を変更することで、パスワード入力欄やメールアドレス入力欄など、様々な形式に変化します。  
また、`<input>`タグでは`<label>`タグを組み合わせて記述して下さい。  
ラベルの文字をクリックしてもチェックボックスや入力欄が反応するため、操作しやすくなります。  
```html
<form>
  <label for="user-name">お名前：</label>
  <input type="text" id="user-name" placeholder="山田 太郎">

  <p>性別：</p>
  <input type="radio" name="gender" id="male" value="male">
  <label for="male">男性</label>
  
  <input type="radio" name="gender" id="female" value="female">
  <label for="female">女性</label>

  <br>
  <button type="submit">送信する</button>
</form>
```  

> **ポイント**  
>`<img>`のsrcや、`<a>`のhrefのように、タグの働きを細かく指定・制御するものを属性と呼びます。属性を使いこなすことで、HTMLタグの役割をさらに引き出すことができます。  

## まとめ
- `<img>`タグで画像を表示し、必ずalt属性で代替テキストを設定する。  
- `<a>`タグは他のページへのリンクを作成する。  
- リストには順序なし(`<ul>`)と順序あり(`<ol>`)があり、各項目は`<li>`で囲む。  
- 表は`<table>` `<tr>` `<th>` `<td>`の役割を理解して組み立てる。  
- ユーザーに入力してもらう要素は`<form>`の中に`<input>`や`<button>`を配置する。
