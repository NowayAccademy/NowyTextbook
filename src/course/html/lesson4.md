# 文章を構成する基本のタグ
前章ではHTMLの構成を解説しましたが、実際にコンテンツを作成するために覚えておきたいタグあります。  
どのようなタグがあるか学習しましょう。

## 本章の目標

本章では以下を目標にして学習します。

- 文章の構成に必要なタグを適切に使えること

## 1. `<h1>～<h6>`  
`<h1>`タグは見出しを作るためのタグです。  
`<h1>`以外にも、`<h2>`、`<h3>`...`<h6>`まであります。 
数字が小さいほど重要度が高く、サイズも大きく表示されます。  
ただし、文字を大きくしたいからという理由で使用することはNGで、重要度に合わせて使用してください。  
また、構造を正しく保つために`<h1>`の次にいきなり`<h3>`を飛ばして使うのは避けましょう。
```html
<h1>もっとも重要な見出し（そのページの「タイトル」）</h1>
<h2>章レベルの見出し（大きな区切り）</h2>
<h3>節レベルの見出し（さらに細かい分類）</h3>
<h4>さらに深い階層の見出し</h4>
```

## 2. `<p>`
`<p>`タグは本文を記述するためのものです。
また、`<p>`タグで段落を表すことができるので`<p>～</p>`を必要な数だけ並べてコンテンツを作成してください。
```html
<p>段落1です。</p>
<p>段落2です。</p>
```

## 3. `<br>`
`<br>`タグは改行を表すタグです。  
`<p>～</p>`の中でも`<br>`タグを記述することで改行することができます。  
`<br>`タグは開始タグ、終了タグはなく  `<br>`と記述するだけで改行すると認識されます。  
```html
<p>ここで<br>改行します。</p>
```

## 4. `<div>`  
`<div>`タグは、コンテンツの一定範囲をひとまとめにするためのものです。 
これ自体は何かを表示するものではないため、このタグで囲んだだけでは表示される内容に影響を与えません。  
一定範囲のスタイルをCSSで一気に変更したい場合に、`<div>`タグでまとめてスタイルを設定したりします。
```html
<div>
  <h1>タイトル</h1>
  <p>本文</p>
</div>
```

## 5. `<span>`  
`<span>`タグは長文の一部を指定する際に使用します。  
このタグも`<div>`タグと同様に囲んだだけでは表示される内容に影響を与えず、スタイルの指定するのに使われます。  

- ### コメント  
プログラミングではソースコードにコメントを残すことができます。  
このコメントはコンピューターは認識せず、Webページには表示されません。  
コメントは人間がソースコードの理解をしやすいように残すものなので、複雑な記述をするときは残すようにしましょう。
```html
<!-- これはコメントです。ブラウザには表示されません -->
```

## まとめ

-  `<h1>～<h6>`タグは文章の「見出し」を定義する。  
- `<p>`タグは段落を表す。  
- `<br>`タグは改行したいときに使用する。
- `<div>`タグと`<span>`タグは特定の範囲をグループ化し、主にCSSでデザインを当てるために使用する。
