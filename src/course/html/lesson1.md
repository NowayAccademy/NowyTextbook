# HTMLとは

HTMLの正式名称は「Hyper Text Markup Language」といいます。  
HTMLを学ぶときによくある勘違いがWebページを「デザインする」ためのものだと考えてしまうことです。  
HTMLは構造を記述するためのもので、どこに何を配置するのかをあらかじめ用意された文法に従って記述します。

# HTMLの要素とタグ
HTMLでは<〇〇>というような記述が多く出てきます。  
これは「タグ」と呼ばれるもので、HTMLの基本的な要素を記述していくためのものです。

HTMLではテキストや写真をはじめ、入力フォーム、テーブル、リストなどさまざ要素を記述するためのタグが用意されています。  
どのような役割のタグがあるかわからないと思い通りのWebページを作ることはできません。  
まずはタグの使い方を覚えましょう。

## タグの記述方法
まずは代表的な<h1>というタグを見てみましょう。  
このタグはタイトルなど「ページに何が書かれているか」を把握するための要素に使用されます。  
この<h1>は以下のようにように「開始タグ」(<○○>)と終了タグ(</○○>)で記述します。  
```java:title
<h1>タイトルです。</h1>
```  

この他に、1つのタグしかないタグもあります。  
例として<br/>という改行するためのタグがあります。
このタグ以下のように<○○>や<○○/>のように記述します。
```java:title
<br> または<br/>
```  


## HTMLの基本構造
それではHTMLの基本的な構造を見てみましょう。  
以下のコードをVSCodeやメモ帳などのテキストエディタで記述して、ブラウザで確認してみましょう。  
保存する時に「ファイル名.html」で保存するとブラウザで開くことができます。
```java:title
<!DOCTYPE html>
<html lang="ja">
<head>
    <meta charset="UTF-8">
    <title>Hello!</title>
</head>
<body>
    <h1>Hello!</h1>
    <p>HTMLの基本構造です。</p>
</body>
</html>
```  
それでは上から順番にそれぞれの役割を見ていきましょう。

### \<!DOCTYPE html>について
この部分はHTMLドキュメントの最初に記述される宣言であり、Webブラウザに対して HTML のバージョンを明示する役割を持ちます。  
現在HTMLの最新バージョンはHTML5です。  
この記述をすることで、Webブラウザに「標準モード」が適用され、HTML5 の仕様に準拠した適切な描画が行われます。  
この記述がない場合にHTML、CSSが正しく表示されない可能性があるので必ずはじめに宣言するようにしましょう。

### \<html lang="ja">について
この部分からHTMLの具体的な内容が始まります。  
\<html>タグから始まり、\</html>タグで終わる形がHTMLの基本形です。  
また「lang="ja"」の部分は属性と呼ばれるもので、そのタグ要素に様々な設定や付加情報を付け加える
ために利用します。  
例えばこのlangは使用する言語を示す属性です。  
"ja"という値を設定することで、このWebページが「日本語」であることを示しています。英語のページであれば"en"と設定します。  
日本語を使う場合は「lang="ja"」をつけるようにしましょう。
