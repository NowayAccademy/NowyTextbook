# HTMLの基本構造
`<!DOCTYPE html>` から `<body>` までの骨組みを解説します。

## 本章の目標

本章では以下を目標にして学習します。

- HTML文書の最小構成を自分で書けること
- `meta charset` の意味を説明できること

## 1. HTMLの基本構造
HTMLの基本的な構造を見てみましょう。  
以下のコードをVSCodeやメモ帳などのテキストエディタで記述して、ブラウザで確認してみましょう。  
保存する時に「ファイル名.html」で保存するとブラウザで開くことができます。  
```html
<!DOCTYPE html>
<html lang="ja">
<head>
  <meta charset="utf-8">
  <title>ページのタイトル</title>
</head>
<body>
  <h1>メインの見出し</h1>
  <p>本文の段落です。</p>
</body>
</html>
```

![基本構造サンプルの出力結果](./images/3-1.png)
ブラウザで確認すると上記のように表示されます。  
それでは上から順番にそれぞれのタグの役割を見ていきましょう。  

## 2. `<!DOCTYPE html>`  
この部分はHTMLドキュメントの最初に記述される宣言であり、Webブラウザに対して HTML のバージョンを明示する役割を持ちます。  
現在HTMLの最新バージョンはHTML5です。  
この記述をすることで、Webブラウザに「標準モード」が適用され、HTML5 の仕様に準拠した適切な描画が行われます。  
この記述がない場合にHTML、CSSが正しく表示されない可能性があるので必ずはじめに宣言するようにしましょう。

## 3. `<html lang="ja">`   
この部分からHTMLの具体的な内容が始まります。  
`<html>`タグから始まり、`</html>`タグで終わる形がHTMLの基本形です。  
また「lang="ja"」の部分は属性と呼ばれるもので、そのタグ要素に様々な設定や付加情報を付け加える
ために利用します。  
例えばこのlangは使用する言語を示す属性です。  
"ja"という値を設定することで、このWebページが「日本語」であることを示しています。英語のページであれば"en"と設定します。  
日本語を使う場合は「lang="ja"」をつけるようにしましょう。

## 4. `<head>`  
`<html>`タグの中に含まれる要素です。  
このヘッダーはWebページに関するさまざまな情報をWebブラウザに伝えるもので表示はされません。  
サンプルでは`<meta>`タグと`<title>`タグが記述されています。

`<meta>`タグは「メタ情報」と呼ばれるものをWebブラウザに付け加えるのに使用します。  
メタ情報とは「画面には表示しないけどWebブラウザに表示する必要がある情報です。  
`<meta charset="utf-8">`の`charset="utf-8"`とは文字をデジタルデータに変換する方式(エンコード)で、「utf-8」を使っていることをWebブラウザに伝えています。**文字化け防止**のためにほぼ必須で使われます。

`<title>`タグはページのタイトルを示すもので、Webブラウザで表示したときに、ウィンドのタイトルバーに表示されます。  
サンプルではタイトルバーに「ページのタイトル」と表示されています。

## 5. `<body>`  
`<body></body>`はWebページに表示するコンテンツを記述するための要素です。  
サンプルでは`<h1>`タグで見出し、`<p>`タグで本文を記述しています。  
このように`<body>`の中でさまざまなタグを組み合わせてWebページを作成します。

> **ポイント**  
>HTMLを記述する際にこの基本構造が必須になるので覚えておきましょう。  
> `<meta charset="utf-8">` は文字化け防止のためほぼ必須です。`<head>` のできるだけ早い位置に書くのが推奨されます。

## まとめ
- `<!DOCTYPE html>` と `html` / `head` / `body` で文書の枠が決まる  
- `meta charset="utf-8"` で**文字コード**を明示する
