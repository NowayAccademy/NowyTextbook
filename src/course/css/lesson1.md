# CSSの適用方法とセレクタ
CSS の読み込み方と、要素を狙い分けるセレクタの基本です。

## 本章の目標

本章では以下を目標にして学習します。

- CSS が HTML の見た目を司ると説明できること
- 外部 CSS ファイルを HTML から読み込めること
- タグセレクタ・クラス・id の違いと、優先度のざっくりした考え方が分かること

## 1. CSS とは

**Cascading Style Sheets** の略で、**レイアウト・色・フォント**などの見た目を指定します。HTML は構造、CSS は**装飾**という役割分担です。

## 2. 外部ファイルの読み込み

```html
<link rel="stylesheet" href="./styles/main.css">
```

`href` に CSS ファイルのパスを書きます。キャッシュや本番配信の話は後の工程で深掘りできます。

> **ポイント**  
> 1ページに複数の `<link>` を置くこともありますが、**ファイル数が増えすぎると読み込み回数**が増えるので、ビルドツールでまとめることもあります。

## 3. セレクタの基本

| 書き方 | 意味 |
| --- | --- |
| `p { }` | すべての `p` 要素 |
| `.title { }` | `class="title"` を持つ要素 |
| `#logo { }` | `id="logo"` の要素（ページ内で原則1つ） |

## 4. 優先度（カスケード）のイメージ

同じプロパティに複数のルールが当たるとき、**より具体的なセレクタ**や **`!important`**（乱用禁止）、**後から書いた宣言**などが優先されることがあります。  
入門では **id < class < 要素** より詳細な指定が強い、とざっくり覚えておけば十分です。

## まとめ

- CSS は**見た目**を担当する  
- `<link>` で**外部 CSS** を読み込む  
- タグ / `.class` / `#id` で**適用先**を指定する
