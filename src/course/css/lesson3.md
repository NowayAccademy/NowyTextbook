# Box Model
文字や色を変えられるようになったら、次は「要素の配置と大きさ」をコントロールします。  
Web上のすべての要素は「四角い箱」として扱われます。  
本章では、この箱の仕組みである「Box Model」を学習します。  

## 本章の目標  

本章では以下を目標にして学習します。  

- Box Modelの4つの領域（Content, Padding, Border, Margin）を説明できること  
- `margin`（外側の余白）と `padding`（内側の余白）を正しく使い分けられること  
- `box-sizing: border-box;` の意味と必要性を理解すること  

## 1. Box Model（ボックスモデル）とは？  
HTMLのすべてのタグ（`<h1>`, `<p>`, `<div>`など）は、目には見えなくても「四角い箱」を持っています。  
この箱は、以下の4つの層で構成されています。  

1. **Content（コンテンツ）**: テキストや画像が表示される、一番中心の領域。  
2. **Padding（パディング）**: コンテンツから境界線までの「内側の余白」。  
3. **Border（ボーダー）**: 箱の「境界線（枠線）」。  
4. **Margin（マージン）**: 境界線の外側にある、他の要素との「外側の余白」。  
これらの調整で要素を配置してデザインします。

## 2. Padding  
`padding`は、箱の内側を広げる余白です。  
背景色は、このPaddingの領域まで塗りつぶされます。  
ボタンをふっくらさせたり、枠線と文字がくっつきすぎるのを防ぐために使います。  

```css  
.button {
  background-color: blue;
  color: white;
  /* 上下左右すべてに20pxの内側余白をつける */
  padding: 20px; 
}

.box {
  /* 上下は10px、左右は30pxの内側余白をつける（よく使う省略記法） */
  padding: 10px 30px; 
}
```

## 3. Border  
`Border`箱の枠線を引きます。
「太さ」「種類（実線や点線）」「色」の3つをセットで指定するのが一般的です。  
```css  
.card {
  /* 1pxの太さで、実線(solid)の、薄いグレーの線を引く */
  border: 1px solid #cccccc;
  
  /* 角を丸くする（これも非常によく使います） */
  border-radius: 8px;
}
```  

## 4. Margin
`margin`は、箱の外側に隙間を作り、他の要素（隣の箱）を押し退けるための余白です。  
背景色は`Margin`には適用されず、常に透明です。  
要素と要素の距離（段落の隙間など）を調整するために使います。  

```css  
h2 {
  /* 下方向にだけ30pxの外側余白をつける */
  margin-bottom: 30px;
}

.container {
  width: 800px;
  /* 左右のmarginをautoにすると、要素が画面の中央に配置されます（頻出テクニック） */
  margin: 0 auto;
}
```  

## まとめ

- Web要素はすべて、`Content`, `Padding`, `Border`, `Margin` の4層の箱でできている。
