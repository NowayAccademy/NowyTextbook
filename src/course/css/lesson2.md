# Box Modelと余白の設定
要素は箱であり、`padding` と `margin` の違いを理解します。

## 本章の目標

本章では以下を目標にして学習します。

- Box Model（content / padding / border / margin）を図で説明できること
- `padding` が内側、`margin` が外側の余白だと区別できること
- レイアウトで余白を取る理由を述べられること

## 1. すべての要素は「箱」

ブラウザは各要素を**長方形の箱**として扱います。

- **content** … テキストや子要素の領域  
- **padding** … **内側**の余白（背景色が伸びることが多い）  
- **border** … 枠線  
- **margin** … **外側**の余白（隣の要素との距離）  

> **ポイント**  
> `box-sizing: border-box` にすると「幅に padding と border を含める」挙動になり、レイアウトが組みやすいことが多いです（リセット CSS で指定されることも）。

## 2. padding と margin

```css
.card {
  padding: 16px;   /* 内側 */
  margin: 0 auto;  /* 外側。ブロックの横中央などに使う例 */
}
```

## 3. デザインでの役割

- **padding** … 文字と枠の間に「呼吸」を作る  
- **margin** … セクション同士の**距離感**を作る  

## まとめ

- 要素は **Box Model** で理解する  
- **内側**は `padding`、**外側**は `margin`  
- 余白は**読みやすさとタップしやすさ**に直結する
