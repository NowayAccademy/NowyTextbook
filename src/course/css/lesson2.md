# 文字と色のスタイリング  
第1章でCSSの基本的な書き方を学びました。  
本章では、Webページのデザインの基本となる「文字」と「色」の整え方を学習します。  

## 本章の目標  

本章では以下を目標にして学習します。  

- 文字色と背景色を指定できること  
- カラーコードの書き方を知ること  
- 文字サイズにおいて、pxとremの違いを説明できること  
- 読みやすい行間の設定方法を理解すること  

## 1. 文字色と背景色を変更する  
要素の色を変更するには、文字色を指定する `color` と、背景色を指定する `background-color` を使います。  

色の指定方法には、主に以下の3パターンがあります。  
1. **カラーネーム**: `red`, `blue`, `black` など決まった名称を記述します。直感的ですが、細かい調整ができません。  
2. **カラーコード（16進数）**: `#ff0000` のように、#と6桁の英数字で表す方法（**実務で最も使われます**）  
3. **RGB / RGBA**: `rgba(0, 0, 0, 0.5)` のように、赤・緑・青の割合と、最後の数値で「透明度」を指定する方法  

```css
/* h1タグの文字を濃いグレーにし、背景を薄い青にする */
h1 {
  color: #333333;
  background-color: #e6f2ff;
}

/* クラス名「note」の背景を半透明の黒にする */
.note {
  background-color: rgba(0, 0, 0, 0.8);
  color: white;
}
```
よく初学者の方は「薄いグレーの背景に、白い文字」のようなオシャレな配色をしがちですが、これでは文字が読めません。  
誰もが読みやすいように、背景と文字の明暗差（コントラスト）をしっかりつけるようにしましょう。  

## 2. 文字のサイズと「単位」  
文字の大きさを変更するには`font-size`を使います。ここで重要なのが**単位**です。  
```css
p {
  /* 絶対的なサイズ（ピクセル） */
  font-size: 16px;
}

h2 {
  /* 相対的なサイズ（レム） ※実務で推奨 */
  font-size: 1.5rem; 
}
```
### px と rem の違いとは？  
- `px`（ピクセル）  
絶対的なサイズです。16pxと指定すれば、どんな環境でも16pxになります。  
- `rem`（レム）  
ブラウザの標準文字サイズ（通常は16px）を「1」とする相対的なサイズです。  
1.5remなら 16px × 1.5 = 24px になります。  
現代の開発現場では、文字サイズに`rem`を使うことが推奨されています。  
なぜなら、視力が弱いユーザーがブラウザの設定で基本の文字サイズを大きくした場合、`rem`で作られたサイトなら設定に合わせて全体が正しく拡大され、ユーザーに優しいページになります。  

## 3. 文字の太さと行間  
文章を「読みやすく」するための重要なプロパティです。  

```css
p {
  font-weight: normal;
  /* 行間を文字サイズの1.6倍にする */
  line-height: 1.6;
}
```
- `font-weight`  
文字の太さを指定します。
normal（標準）や bold（太字）で指定します。  
- `line-height`  
行と行の隙間を調整します。  
文章の読みやすさに直結する非常に重要なプロパティです。 
`line-height`に単位は書かずに数値を指定します。  
line-height: 24px; のように固定値で書くと、後から文字サイズを大きくした時に、行同士が重なってしまうバグが発生します。 
1.5や1.7のように**単位をつけない数値（倍率）**で指定するのが、保守性の高い安全な書き方です。  

## 4. テキストの配置と装飾  
文字の寄せ方と、線の装飾について学びます。  
```css
h1 {
  /* 文字を中央揃えにする（left, right, center がある） */
  text-align: center;
}

a {
  /* リンクの下線を消す */
  text-decoration: none;
}
```
- `text-align`  
文字の寄せ方を指定します。  
`center`と指定すれば真中寄せ、`right`と指定すれば右寄せになります。  

- `text-decoration`
`<a>`タグはデフォルトで下線がついてますが`text-decoration: none`とスタイルを設定するとなくなります。
昨今では、下線がないほうが一般的なので覚えておきましょう。

## まとめ

- `color`は文字色、`background-color`は背景色を指定する。色は16進数（#）での指定が一般的。  
- `font-size`の単位には、ユーザーの環境に配慮できる`rem`を積極的に使う。  
- 文章の読みやすさは`line-height`で決まる。指定時は`px`などの単位をつけず、倍率のみで書く。  
- `<a>`タグのデフォルトの下線は`text-decoration: none;`で消すことができる。
