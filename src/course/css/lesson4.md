# CSSハンズオン
配布ページにスタイルを追記し、質疑応答で仕上げます。

## 本章の目標

本章では以下を目標にして学習します。

- 外部 CSS に**セレクタと宣言**を追加できること
- Box Model と Flexbox を**実際の画面**で確認できること
- うまく当たらないときに**開発者ツールで検証**できること

## 1. 進め方（想定）

- **0から一から作らず**、配布 HTML/CSS に**追記・調整**する形式を想定  
- セレクタの当たり先を Elements で確認しながら進めます  

> **ポイント**  
> 「どの要素にどのルールが当たっているか」が分かると、**打ち消し線**の意味も読み取れるようになります。

## 2. 試すとよいテーマ例

- カードに `padding` と `box-shadow`  
- ナビを `flex` で横並び・右寄せ  
- `gap` でボタン間の余白  

## 3. 質疑応答

想定では**約5分**を質問に充てます。  
当たらないルールは、**セレクタのスペル**と**詳細度**を疑いましょう。

## まとめ

- ハンズオンは**追記型**で時間内に仕上げやすい  
- DevTools で**適用ルール**を見る習慣をつける  
- Flexbox は**配置の時短**に効く
