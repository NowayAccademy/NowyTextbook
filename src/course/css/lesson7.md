# レスポンシブデザインとメディアクエリ
現代のWebサイトは、小さなスマートフォンから巨大なPCモニターまで、様々な画面サイズで閲覧されます。  
それぞれの画面幅に合わせてレイアウトを自動的に変化させる仕組みを「レスポンシブデザイン」と呼びます。  
本章では、その要となる技術「メディアクエリ」を学習します。  

## 本章の目標

本章では以下を目標にして学習します。  

- 画面サイズに応じてCSSを切り替える `@media` の書き方を理解すること  
- ブレイクポイント（切り替えの基準点）の概念を知ること  
- 実務の標準である「モバイルファースト」の考え方を説明できること  
- はみ出す画像を防ぐ `max-width: 100%;` を使いこなせること  

## 1. メディアクエリ（`@media`）の基本  
特定の画面サイズ（条件）のときだけ適用されるCSSを書くためのルールが、メディアクエリ（`@media`）です。  

一番簡単な例を見てみましょう。  
```css
/* ① 基本のCSS（すべての画面に適用される） */
.box {
  background-color: blue;
}

/* ② 画面幅が「768px以上」のときだけ適用されるCSS */
@media (min-width: 768px) {
  .box {
    background-color: red; /* 青色が赤色に上書きされる！ */
  }
}
```  
このように、`@media`のブロックの中に書いたCSSは、条件（今回は「横幅が768px以上」）を満たした時にだけ発動し、基本のCSSを上書きします。  

## 2. 実務の鉄則「モバイルファースト」 
メディアクエリを書く際、条件の指定方法には`min-width`（〇〇px以上）と`max-width`（〇〇px以下）の2種類があります。  
- PCファースト  
最初にPC用のCSSを書き、`@media`(max-width: 768px) で「スマホの時はこうする」と後から変更する手法。  
- モバイルファースト:
最初にスマホ用のCSSを書き、`@media`(min-width: 768px) で「PCなど画面が広くなったらこうする」と追加していく手法。  

現代の開発現場では、圧倒的に**モバイルファースト**が推奨されています。  
スマホ用のレイアウト（基本は1列に縦並び）はシンプルで記述量が少なく済みますが、PC用のレイアウトは複雑になりがちです。  
複雑なPCのCSSをスマホ用に打ち消していく作業はバグの温床になるため、「シンプルなスマホ用を基準にし、広い画面には装飾を足していく」という考え方が合理的だからです。  

## 3. Flexboxとの連携（よくあるレイアウト切り替え）  
実務で最もよく書くメディアクエリは、「スマホの時は縦並び、PCの時は横並び」という`Flexbox`の切り替えです。  
```css  
/* スマホ用の基本レイアウト（縦並び） */
.card-container {
  display: flex;
  flex-direction: column; /* 縦に並べる */
  gap: 20px;
}

/* タブレット・PC用のレイアウト（横幅768px以上のとき） */
@media (min-width: 768px) {
  .card-container {
    flex-direction: row; /* 横に並べる */
  }
}
```  

## 4. はみ出す画像の制御（可変レイアウト）  
画面幅が変化するレスポンシブデザインにおいて、もう一つ絶対に覚えておくべきプロパティがあります。  
大きな画像（例えば横幅1000pxの写真）をスマホ（横幅390pxなど）で表示すると、画面を突き抜けて横スクロールが発生してしまいます。  
これを防ぐのが以下の魔法のコードです。  
```css  
img {
  max-width: 100%; /* 親要素（画面）の幅を超えないように縮む */
  height: auto;    /* 縦横比を維持する */
}
```  
この指定をしておくだけで、どんなに大きな画像でも、スマホの画面幅に合わせて自動的に縮んでくれるようになります。  

## まとめ

- 画面幅に応じてCSSを切り替えるには`@media`（メディアクエリ）を使用する。  
- 現代の標準は、スマホ用を基本として書き、`min-width`でPC用を足していく「モバイルファースト」。  
- スマホでは縦並び、PCでは横並びというFlexboxの切り替えが実務で多用される。  
- 画像には`max-width: 100%;``height: auto;`を指定し、画面からはみ出さないようにする。
