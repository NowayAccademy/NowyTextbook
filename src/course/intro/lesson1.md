# 初回レッスン（導入）
受講の進め方と、卒業時に目指す姿を共有します。

## 本章の目標

本章では以下を目標にして学習します。

- 講座の流れとサポートの受け方が分かること
- 卒業時に達成したいゴールを言葉にできること
- 質問・相談の出し方が分かること

## 1. はじめに

教材は**コースごと**に分かれています（**初回レッスン**、HTML、CSS、JavaScript、画面**設計**、Git、**TypeScript/React** など）。このフォルダは**初日の導入と環境**に使います。  
初日は「全体像」と「学習の土台」を固める時間です。

> **ポイント**  
> 分からないことは早めに共有するほど、後戻りが減ります。「恥ずかしい」と思わず、講師・メンター・受講仲間を活用しましょう。

## 2. あいさつと自己紹介

受講者同士・講師との顔合わせを行います。  
名前、学習・仕事の背景、**このコースで特に身につけたいこと**を短く話せるようにしておくとスムーズです。

## 3. 卒業時の目標の共有

卒業時に目指す状態の例は次のようなイメージです（受講生・講師で具体化します）。

- HTML/CSS で**静的なページ**を自分の力で組み立てられる
- JavaScript で**DOM 操作・イベント**の基本が書ける
- Git で**ブランチ・コミット・PR**の一連ができる
- React（TypeScript）で**小さなコンポーネント**を組み合わせた画面が作れる

> **ポイント**  
> ゴールは人によって異なります。**「自分は何ができるようになりたいか」**を一文で書いておくと、迷ったときの指針になります。

## 4. 1日目のスケジュール感

初日は、本フォルダの**初回・環境**に加え、**HTML / CSS / JavaScript** コースの導入が続く想定です。  
詳細な時間配分は当日の資料に従ってください。

## まとめ

- 初回は**関係構築**と**ゴールの言語化**が主目的である  
- 分からない点は**早めに質問**する習慣をつける  
- 卒業時のイメージを**自分の言葉で持つ**
