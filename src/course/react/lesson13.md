# Reactハンズオン（質疑応答含む）
小さなコンポーネントを組み合わせ、props / state を体感します。

## 本章の目標

本章では以下を目標にして学習します。

- 配布または雛形プロジェクト上で**コンポーネント分割**ができること
- **props** で表示を変え、**useState** でクリックなどに反応できること
- つまずきを**型エラーと実行時エラー**に分けて説明できること

## 1. 想定内容

- 子コンポーネントに **props** を渡す  
- `useState` でトグルやカウンタ  
- **条件付きレンダリング**でメッセージ切り替え  

> **ポイント**  
> エラーは**コンソール**と**エディタの赤線**の両方を見る。メッセージをそのまま検索すると公式ドキュメントに当たることが多いです。

## 2. 質疑応答

想定時間内に**質問・デバッグ**の枠を取ります。再現手順とエラー全文があると助かります。

## まとめ

- 手を動かして **props / state / 条件表示**を1本の流れで体験する  
- 型と実行時の**両方**でデバッグ視点を持つ  
- 小さく動いてから**機能を足す**
