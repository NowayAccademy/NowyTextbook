# JavaScriptハンズオン
DOM 操作・条件分岐・ループを組み合わせて実装し、質疑応答で仕上げます。

## 本章の目標

本章では以下を目標にして学習します。

- 配布コードに沿って**要素の取得と更新**ができること
- `if` と `for` を使った**簡単な一覧処理**ができること
- つまずきを**質問に変換**できること

## 1. 進め方（想定）

- **DOM 操作**（テキスト差し替え、リストへの追加など）  
- **if 文**（入力値や状態に応じた表示切り替え）  
- **for 文**（配列データのループ表示）  

想定時間に**質疑応答**を含みます。エラーは Console のメッセージをそのままメモしておくと解決が早いです。

> **ポイント**  
> 「動いたら終わり」ではなく、**1行ずつ何をしているか**を声に出せるとレベルアップが早いです。

## 2. つまずきやすいポイント

- `getElementById` の **id スペルミス**  
- スクリプト実行時点で**まだ DOM に要素がない**（読み込み順）  
- ループの**終了条件**（`<= length` など）  

## まとめ

- ハンズオンは **DOM + if + for** の三本柱である  
- Console と Sources を**併用**すると効率が上がる  
- 質問は**再現手順**付きで出す
