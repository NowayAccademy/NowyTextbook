# イベント処理によるインタラクション
クリックなどのイベントを受け取り、処理を実行します。

## 本章の目標

本章では以下を目標にして学習します。

- イベント駆動で UI が動くイメージを説明できること
- `addEventListener("click", ...)` の基本形が書けること
- 無名関数とアロー関数のどちらでも登録できること

## 1. イベントとは

クリック、入力、キー操作、読み込み完了など、**ブラウザが通知する出来事**です。  
JavaScript はそれを**待ち受け（リスン）**して処理を走らせます。

## 2. addEventListener

```html
<button id="btn" type="button">押す</button>
<script>
  const btn = document.getElementById("btn");
  btn.addEventListener("click", () => {
    console.log("クリックされました");
  });
</script>
```

第1引数はイベント名（文字列）、第2引数は**コールバック関数**です。

> **ポイント**  
> HTML の `onclick` 属性でも書けますが、**JS 側に寄せる**と保守しやすいことが多いです。

## 3. よく使うイベント（例）

| イベント | タイミングの例 |
| --- | --- |
| `click` | クリックされた |
| `input` | 入力欄の値が変わった |
| `change` | 選択が確定した |
| `submit` | フォーム送信（キャンセルもできる） |

## まとめ

- UI の「動き」は**イベント → 処理**の連鎖である  
- `addEventListener` で**種類と関数**を登録する  
- まずは **`click`** に慣れると応用しやすい
